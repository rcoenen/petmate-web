# PETSCII Compo 25

These files are from the CSDb event:
[Plain PETSCII Graphics Competition 2025 - For the Honour of PETSCII](https://csdb.dk/event/?id=3527)

The artwork was extracted using Retro Debugger, exported as `.vce`, and imported into Petsciishop using a `.vce` importer added to Petsciishop.

After import, everything was exported to Petsciishop's default `.sdd` format and saved in this folder.
